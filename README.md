# Cassette Player Py

Um toca-fitas retrô desenvolvido em Python com interface gráfica inspirada nos aparelhos clássicos dos anos 80 e 90.

## Recursos

- Reprodução de áudio
- Botões Play, Stop, Pause
- Efeitos sonoros mecânicos
- Visual retrô
- Simulação de fita cassete

## Instalação

```bash
pip install -r requirements.txt
python main.py
